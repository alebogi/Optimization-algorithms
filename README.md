# Optimization algorithms

Homeworks for Engineering Optimization Algorithms course.

Every repository contains pdf file in Serbian. At the end of the file are informations about that project.

## Description

- Project 1 (domaci1): Brute force algorithm
- Project 2 (domaci2): Brute force algorithm for problems in TSP class
- Project 3 (domaci3): Brute force - spanning tree
- Project 4 (domaci4): Spherical Bessel function and bisection algorithm
- Project 5 (domaci5): Nelder-mead simplex algorithm
- Project 6 (domaci6): Linear programming
- Project 7 (domaci7): Simulated annealing algorithm
- Project 8 (domaci8): Genetic algotithm
- Project 9 (domaci9): Differential evolution algorithm
- Project 10 (domaci10): Pso algorithm
- Project 11 (domaci11): Pareto front

## Built with
- Projects 1, 2 and 3: C++
- Other: Python
